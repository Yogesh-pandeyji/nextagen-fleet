# Nextagen Fleet Platform

Production fleet management SaaS. Tech: React 18, Vite, Tailwind, Zustand, Supabase.

## Quick Start

1. Clone repo and run `npm install`
2. Copy `.env.example` to `.env` and add your Supabase credentials
3. Run `npm run dev`

Deploy to Vercel: import repo, add env vars, deploy.
